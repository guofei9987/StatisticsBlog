def vif(df, col_i):
    cols = list(df.columns)
    cols.remove(col_i)
    cols_noti = cols
    formula = col_i + '~' + '+'.join(cols_noti)
    r2 = ols(formula, df).fit().rsquared
    return 1. / (1. - r2)

5. ElasticNet¶

官网
$\alpha \rho \mid\mid \vec w\mid\mid+\dfrac{\alpha (1-\rho)}{2}\mid \mid \vec w \mid \mid^2$
$\alpha\geq 0, 1\geq \rho \geq 0$
